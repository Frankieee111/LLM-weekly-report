# AI大模型周报2025年10月第一期 技术整理Markdown笔记
### [Sora 2] https://openai.com/index/sora-2
OpenAI推出音视频生成模型Sora Two，如果说2024年2月的初代SORA是视频模型的GPT1时刻，那么Sora Two也许是GPT3.5时刻，能够更好地遵循物理法则，支持更精细、时间跨度更久的指令，擅长纪实、电影、动画等多种风格。Sora Two可以模拟成功以及失败的动作，支持生成复杂且真实的音乐、音效、人声，另外模型支持无缝的主体嵌入，比如基于一段视频复刻一个人、一个动物或者一件物品的外形和声音，并将其自然地嵌入任何生成的环境中。

### [OpenAI on OpenAI] https://openai.com/index/building-openai-with-openai
OpenAI介绍了OpenAI on OpenAI系列，该系列展示公司如何在真实业务中使用AI解决问题，其中包括GTM Assistant，也就是Go to Market Assistant，可辅助项目研究、会议，准备产品问答等等，能显著提升销售成果；ChatGPT将合同转化成结构化、可搜索的数据，赋能规模化的合同审核；还有Inbound Sales Assistant为主动问询的潜在客户，即刻提供关于产品与合规问题的个性化回复，连通完整的背景信息，对接销售人员等等。

### [GDPval基准] https://openai.com/index/gdpval
OpenAI推出GDP Value新基准评估模型，通过模型在对GDP有重要贡献的九个行业、44种职业的1320项专门任务中的表现，衡量AI模型实际的经济价值。初步结果显示，Cloud Opus4.1表现最佳，尤其擅长文件排版等美学的任务；GPT5尤其擅长专业知识、搜索等关乎准确性的任务；相较人类行业专家，前沿模型能够以100倍的速度和1%的成本，以及相近的质量完成GDP相关的任务。

### [Fal] https://fal.ai
Flow是由Google Cloud驱动的一站式媒体内容生成平台，帮助开发者为应用快速融入图像、音视频、声音生成等功能，一个API即可接入200多个模型，面向商务、零售、游戏等领域，能创建个性化的实时体验。

### [Lovable Cloud & AI] https://lovable.dev/cloud
LOVEBOOK推出Loveable Cloud和LOVEABLE AI，只需提示词即可发布全站式AI原生的各类应用，比如社交软件、社区平台、库存系统、协作工具等等；当用户增加、流量激增的时候，用户可以随时升级Loveable Cloud实例，以满足需求。

### [HunyuanImage3.0] https://hunyuan-image.com
腾讯推出多模态图像生成模型混元EMH3.0，结合扩散架构与增强压缩技术，同时完善双编码器系统，支持新一代人类反馈增强学习优化，实现高级美学，支持多语种，能以更低的计算成本，高效生成高质量图像。

### [KIMI智能体模式（OK Computer）] (无对应链接)
KIMI推出智能体模式，OK Computer依托Kimi K2可以作为产品经理、战略规划师、UI设计师、代码工程师等执行任务，单次可以处理100万行数据，还能生成图表、PPT等，有网友认为它比MAC AI好用。

### [Claude Sonnet 4.5] https://www.anthropic.com/news/claude-sonnet-4-5
Anthropic推出Claude Sonnet 4.5软件，代码能力优秀，尤其适合创建复杂智能体，擅长调用工具解决计算机相关任务，在推理和数学领域提升显著；相较此前模型，其在金融、法律、医药、STEM等学科的专门领域，知识和推理能力都更优秀。

### [RoboBrain-X0] https://github.com/FlagOpen/RoboBrain-X0
北京智源研究院推出跨本体基础模型，也就是Cross Ontology Foundation Model Robot Brain X0，能够克服单一机器人系统的限制，支持异构本体迁移，比如Adept XR1 Light双臂轮式机器人、还有FRANKA单臂系统等等；具备高效的零样本泛化能力和复杂任务解构能力，具身推理任务性能领先。

### [RecoWorld] https://arxiv.org/pdf/2509.10397
Meta面向智能体推荐系统，推出Record World双视角环境模拟框架A Duo View Architecture，让模拟用户与智能体推荐系统以用户流程最大化为目标，展开多轮交互；在不影响真实用户体验的同时，让智能体推荐器根据指令和推理调整推荐内容，形成主动留住用户的动态反馈闭环，这样推荐系统不再只是单向的，而是由用户和智能体协作塑造个性化的信息流；Record World支持多智能体模拟，也就是Multi Agent Simulations，可以模拟特定用户群的反应。

### [Modular Manifolds] https://thinkingmachines.ai/blog/modular-manifolds
Thinking Machines发布论文Modular Manifold，将微分几何中的流形理论引入神经网络优化，提出模块化流形框架，也就是Modular Manifold，让权重张量在预定义的流形约束下进行优化，从而在保证理论严谨性和网络稳定性的同时，提升训练效率。

### [DeepSeek-V3.2-Exp] https://api-docs.deepseek.com/news/news250929
深度求索发布DeepSeek V3.2 Experimental模型，在V3.1 Terminus的基础上，引入稀疏注意力机制DeepSeek Sparse Attention，在几乎不影响模型输出效果的前提下，大幅提升长文本训练和推理效率，同时API调用成本降低50%以上。

### [GEMINI 2.5 Flash/2.5 Flash Light] (无对应链接)
Google DeepMind发布GEMINI 2.5 Flash和2.5 Flash Light更新版，质量和速度都有提升，输出Tokens效率更高，继而成本大幅下降。

### [Toyota Woven City] (无对应链接)
2020年发起的智慧城市项目Toyota Woven City近日宣布，正式开始入驻居住者，主要包括Inventors和Residents两类人；前者主要包括企业经营者、初创公司、研究机构、艺术家等创新者，他们将在这里开发测试自己的产品和服务，由居民以及游客，也就是Residents来进行一手体验并给予反馈；预计第一阶段将入住300名居民，2026财年开始面向广大游客开放。

### [国星宇航太空计算星座] (无对应链接)
中国国星宇航实现太空计算星座的常态化商业运营，该星座由首批星座卫星组成，旨在构建由2800颗计算卫星组成的天机算力基础设施，总算力超过10万PetaFLOPS，支持E级参数模型运行；在第四届八闽算法大赛上，佳都科技旗下团队提交了基于这个太空计算星座的解决方案，实现星上模型部署。

### [微软微流体冷却系统MICROFLUIDX] (无对应链接)
微软推出新型冷却系统，使用微流体冷却技术MICROFLUIDX，通过在芯片上直接蚀刻微通道，冷却液可以准确流向CPU和GPU的发热核心区域，实现冷板冷却，拥有三倍的散热性能，可以将GPU内部硅片的最大升温降低65%；这类架构能够减少水和能源的消耗，帮助构建更加可持续、高性能的数据中心。

# AI大模型周报2025年10月第二期 
### [Gemini 2.5 Computer Use] https://blog.google/technology/google-deepmind/gemini-computer-use-model
Google DeepMind推出GEMINI 2.5 Computer Use模型，该模型基于GEMINI 2.5 Pro的视觉理解和推理能力构建，可赋能智能体实现与用户界面的交互，在多个网页和移动端控制基准测试中表现领先，且具备更低的延迟。开发者可通过Google AI Studio和Vertex AI的Gemini API获取该模型能力，适用于网页浏览器交互场景，在移动端UI控制任务中也展现出较强潜力。

### [Opal] https://opal.withgoogle.com/landing
Google Labs推出的Opal零代码AI迷你应用创建服务，已在美外新增覆盖15个国家，同时新增工作流故障排除（Advanced Debugging for Workflows）等功能，支持用户通过自然语言快速构建功能强大的AI迷你应用，无需编码即可完成从创意到可用应用的转化，还提供直观的节点式工作流编辑功能，方便用户优化应用逻辑。

### [Apps in ChatGPT] https://openai.com/index/introducing-apps-in-chatgpt
OpenAI推出Apps in ChatGPT功能，用户可在ChatGPT聊天界面直接与第三方应用进行交互。首批支持的应用包括Booking.com、Canva、Coursera、Figma、Expedia、Spotify和Zillow，2025年还将上线11款新应用；同时同步推出基于Model Context Protocol（MCP）的Apps SDK预览版，帮助开发者便捷设计应用的逻辑与界面，该SDK为开源标准，支持在任何采用该标准的平台运行。

### [OpenAI AgentKit] https://openai.com/index/introducing-agentkit
OpenAI推出AgentKit工具集，专为开发者和企业打造，用于高效创建、部署和优化智能体。该工具集包含Agent Builder（可视化多智能体工作流设计画布）、Connector Registry（数据与工具连接管理中心）、ChatKit（嵌入式聊天界面工具包）等核心组件，同时扩展了数据集管理、追踪评分、自动提示词优化、第三方模型支持等评估功能，可助力提升智能体开发效率与性能。

### [Cursor Plan Mode] https://cursor.com/cn/blog/plan-mode
Cursor推出Plan模式，在处理复杂编程任务时，模型可先分析用户的代码库，查找相关文件、阅读文档并主动提出澄清性问题，进而生成包含文件路径和代码引用的详细计划。用户可直接编辑该计划（包括添加或删除待办事项），也可将其保存为Markdown文件供后续参考，该模式能显著提升生成代码的质量，当用户描述复杂任务时，Cursor会自动建议启用该模式。

### [Claude Code Plugins] https://www.anthropic.com/news/claude-code-plugins
Anthropic公开测试Claude Code Plugins功能，支持将Slash Commands（自定义快捷指令）、Subagents（子智能体）、MCP servers（模型上下文协议服务器）、Hooks（工作流钩子）等多种扩展组件打包为可共享模块，用户可通过`/plugin`命令在Claude Code中直接安装使用。插件支持按需启用或禁用，可降低系统提示上下文复杂度，同时支持搭建插件市场，方便开发者分享和发现各类定制化插件。

### [Ling-1T] https://zenmux.ai/inclusionai/ling-1t?utm_source=hf_inclusionAI
蚂蚁集团前沿AI研究团队Inclusion AI发布稀疏混合专家通用语言模型Ling-1T，该模型参数规模达1万亿，每token活跃参数为500亿，在多个复杂基准测试中表现领先，尤其擅长代码生成、软件开发、专业数学逻辑推理等任务，兼具高准确性和运行效率；平台当前提供充值额外赠送10%额度的活动，邀请好友还可获得专属礼品。

### [NeuroChat] https://www.media.mit.edu/projects/neurochat-ai-bci/overview
MIT Media Lab研发的NeuroChat是一款神经自适应AI聊天机器人，融合生成式AI与实时脑电图（EEG）数据，可打造个性化动态学习体验。该机器人无需依赖用户明确输入，而是通过EEG设备捕捉大脑信号，推断用户的参与度、认知状态和学习偏好，当检测到用户参与度下降时，会自动调整对话策略，包括优化响应复杂度、节奏和交互程度，以重新吸引用户注意力，目前仅支持Muse EEG头戴设备，适用于自主学习者、知识工作者等群体。
